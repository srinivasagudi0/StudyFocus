# StudyFocus 📚

StudyFocus is a simple app designed to help you track your study sessions and understand your learning patterns. StudyFocus keeps you on track and motivated. It has a cool authentication and stores data privately. It also uses OpenAI API to analyze how you study based on your past study sessions. 

It helps you understand the way you study and keeps motivating you to log more study time. It is really easy to use also. It is made of python, HTML and some CSS.

This is also my first Flask app. Hope you guys like it. ❤️
